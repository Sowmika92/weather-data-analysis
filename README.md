# Weather Data Analysis Project

This project demonstrates a complete exploratory data analysis (EDA) for a weather dataset, including at least two different chart types and written insights.

Contents
- data/weather_sample.csv — small sample dataset
- analysis/analysis.py — Python script to run the analysis, generate charts and a report
- analysis/analysis_report.md — written insights (also generated by the script)
- requirements.txt — Python dependencies

Quick start
1. Create a virtual environment (recommended)
   python3 -m venv venv
   source venv/bin/activate   # macOS/Linux
   venv\Scripts\activate      # Windows

2. Install dependencies
   pip install -r requirements.txt

3. Run the analysis
   python analysis/analysis.py

4. Output
   - analysis/outputs/ contains PNG charts:
     - temp_timeseries.png (line chart of temperatures)
     - temp_histogram.png (histogram + KDE of mean temperature)
     - temp_boxplot_by_month.png (boxplot of mean temp by month)
     - corr_heatmap.png (correlation heatmap)
     - temp_vs_humidity_scatter.png (scatter + regression)
   - analysis/analysis_report.md — written insights and summary statistics

Using your own dataset
- If you have your own CSV, place it at data/weather_sample.csv or edit the script to point to your file. The script expects these columns:
  - date (YYYY-MM-DD)
  - temp_max
  - temp_min
  - temp_mean
  - humidity (percentage)
  - wind_speed (m/s or km/h)
  - precipitation (mm)
  - pressure (hPa)

Notes
- The sample dataset is small and synthetic to allow quick demonstration.
- The script performs basic cleaning (date parsing, missing value handling), EDA, and saves charts and a textual report.
